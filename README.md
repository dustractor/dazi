#dazi

pronounced dot see

a filetype-plugin for vim

filetype is dot

be editing a dot

push button make see it in feh

to have button one of two ways accomplish:

make a mapping to ``Dazi`` command

    nnoremap <blah> :Dazi<CR>

or set global ``dazimap`` to the key you want

    let g:dazimap = '<blah>'
